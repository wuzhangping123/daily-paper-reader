## 摘要
光场（LF）图像去噪由于LF数据的高维结构而具有挑战性。虽然噪声在子孔径图像中是独立的，但场景内容表现出强烈的跨视图相关性。我们引入了VSANet，一种用于LF去噪的视图感知稀疏注意力网络。具体来说，我们提出了一个视图感知稀疏注意力（VSA）块，它将4D LF特征图表示为统一的空间-角度标记空间，并通过基于局部敏感哈希的稀疏注意力进行跨视图聚合。这使得全局特征交互具有线性复杂度，有效地利用了跨视图和空间位置的LF相关性。此外，我们设计了一个特征细化（FR）块，以强调空间、角度和对极子空间中的信息性特征。VSA和FR块被集成到一个顺序注意力细化模块中，构成了VSANet的核心。实验表明，VSANet优于最先进的LF去噪方法。

## Abstract
Light field (LF) image denoising is challenging due to the high-dimensional structure of LF data. While noise is independent across sub-aperture images, scene content exhibits strong cross-view correlations. We introduce VSANet, a view-aware sparse attention network for LF denoising. Specifically, we propose a view-aware sparse attention (VSA) block that represents the 4D LF feature map as a unified spatial-angular token space and performs cross-view aggregation via locality-sensitive hashing-based sparse attention. This enables global feature interactions with linear complexity, effectively exploiting LF correlations across views and spatial locations. In addition, we design a feature refinement (FR) block to emphasize informative features in spatial, angular, and epipolar subspaces. The VSA and FR blocks are integrated within a sequential attention refinement module, forming the core of VSANet. Experiments demonstrate VSANet outperforms stateof-the-art LF denoising methods.

- **研究动机**：利用LF信号与噪声的不对称性（噪声独立、场景相关），设计一种能够高效进行全局非局部聚合的轻量级网络，同时保持计算可行性。
- **整体含义**：提出VSANet，首次在LF去噪中实现对整个4D LF表示进行注意力操作，通过局部敏感哈希（LSH）稀疏注意力实现线性复杂度的全局跨视角特征交互，并设计特征细化模块增强多子空间判别表示，从而在基准数据集上超越现有最先进方法。

## 2. 论文提出的方法论：核心思想、关键技术细节
### 核心思想
- 将整个4D LF特征图视为统一的**空间-角度令牌序列**，直接在所有视角和空间位置上执行非局部注意力，实现全局交互。
- 利用LSH将相似令牌分组到同一哈希桶，在桶内局部窗口内计算注意力，将复杂度从O(N²)降至O(N·s)（N=令牌总数，s=块大小）。
- 设计顺序提取注意力细化（SEAR）模块，集成**视点感知稀疏注意力（VSA）块**和**特征细化（FR）块**。

### 关键技术细节
1. **网络总体架构**（图1）：
   - 输入噪声LF Ln → 空间子空间(SS)中卷积浅层特征提取 → 重塑为4D表示（R4D操作） → 堆叠G个SEAR模块（带全局残差连接） → 卷积输出 → 重塑得到去噪LF Ld。
2. **SEAR模块内部**：
   - **顺序特征提取（SFE）**：F个级联特征提取（FE）块。每个FE块含两个并行分支：
     - 空间-角度分支：在SS和角度子空间(AS)中操作，捕获空间和角度相关性。
     - 对极分支：在水平对极子空间(EHS)和垂直对极子空间(EVS)中操作，捕获几何一致性。
     - 两分支特征融合后送入VSA块。
   - **VSA块**（核心）：
     - 对LF特征图Lf用两个并行卷积生成查询Q、键K（维度Cr=C/r）和值V（维度C）。重塑为统一令牌序列（长度N=S·T·H·W）。
     - 采用LSH稀疏注意力：进行nh次独立哈希轮次（HR），每轮生成随机旋转矩阵R^(k)∈R^{Cr×⌊Bh/2⌋}，令牌按式(1)分配到Bucket。同一Bucket内令牌排序后分割为大小为s的非重叠块，每个块扩展邻接块形成3s上下文窗口，在窗口内计算缩放点积注意力（式(2)）。输出通过置信度加权聚合（式(3)）融合各轮结果，其中πk基于log-sum-exp归一化。
     - 聚合特征重塑后通过残差连接添加回Lf。
   - **FR块**：在SS、AS、EHS、EVS四个子空间中分别应用CBAM（通道注意+空间注意），增强判别特征，然后将四个子空间特征拼接、卷积融合，送入下一SEAR模块。

### 公式说明（文字）
- LSH分桶：对每个令牌的查询/键向量与随机旋转矩阵的乘积及其负值连接，取argmax得到bucket索引。
- 注意力权重：在局部窗口内计算softmax归一化点积。
- 多轮聚合：使用每个哈希轮次的log-sum-exp作为置信度，加权平均各轮输出。

## 3. 实验设计
