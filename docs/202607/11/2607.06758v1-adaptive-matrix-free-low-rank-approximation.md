## 摘要
我们研究无矩阵环境下的固定容差低秩近似问题，其中矩阵或线性算子$\mathbf{A}$只能通过矩阵-向量乘积访问，且其秩必须自适应地确定以满足预设的误差容限。我们引入了一族自适应、无矩阵的随机化QB算法。随机误差指示器直接从随机草图估计残差范数（无论是Frobenius范数还是谱范数），精度可达到机器精度。无矩阵的秩剪枝步骤将计算块大小与最终秩解耦，因此可以使用大型、BLAS-3友好的块而不会过度估计秩；同时，一种无伴随的变体仅使用正向算子返回标准正交基。在具有不同奇异值衰减的测试矩阵上，所提出的方法获得的秩接近截断SVD最优值，同时以高概率满足预设容差。

## Abstract
We study fixed-tolerance low-rank approximation in the matrix-free setting, where a matrix or linear operator $\mathbf{A}$ is accessible only through matrix-vector products and its rank must be determined adaptively to meet a prescribed error tolerance. We introduce a family of adaptive, matrix-free randomized QB algorithms. A randomized error indicator estimates the residual norm -- in either the Frobenius or the spectral norm -- directly from a random sketch, remaining accurate down to machine precision. A matrix-free rank-pruning step decouples the computational block size from the final rank, so that large, BLAS-3-friendly blocks can be used without over-estimating the rank, and an adjoint-free variant returns the orthonormal basis using only the forward operator. Across test matrices with diverse singular-value decays, the proposed methods attain ranks close to the truncated-SVD optimum while meeting the prescribed tolerance with high probability.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：在**无矩阵（matrix-free）** 环境下，矩阵或线性算子 $\mathbf{A}$ 只能通过**矩阵-向量乘积（matvec）** 访问，无法直接获取其元素。针对固定容差（fix-tolerance）的低秩近似问题，需要**自适应地确定秩（rank）**，使得近似误差 $\|\mathbf{A} - \mathbf{Q}\mathbf{B}\| \leq \tau$（通常 $\tau = \varepsilon$ 或 $\varepsilon \|\mathbf{A}\|$），其中 $\mathbf{Q}$ 具有正交列，$\mathbf{B} = \mathbf{Q}^\top \mathbf{A}$。
- **研究动机**：现有方法存在三大瓶颈：
  - 经典确定性方法（如SVD、RRQR）需显式矩阵条目，无法在无矩阵下应用。
  - 随机化固定秩方法（如随机SVD、Nyström）需预先指定秩，不适用于秩动态变化的场景（如分层矩阵构造）。
  - 现有自适应方法分为两种：
    - **Frobenius范数**下的 `randQB_EI`（Yu et al. 2018）使用 Gramian 减法跟踪残差，存在**精度墙**（约 $\sqrt{\varepsilon_{\text{mach}}} \approx 10^{-8}$），且需显式计算 $\|\mathbf{A}\|_F$，非无矩阵。
    - **谱范数**下的 `randQB_HMT`（Halko et al. 2011）使用保守概率界，导致**秩过度估计**。
- **整体含义**：填补无矩阵、自适应、高精度低秩近似方法的空白，为大规模隐式算子（如PDE Hessian、积分方程）的压缩提供高效工具。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：在逐块随机化QB框架基础上，利用**随机草图（random sketch）** 同时完成：
  - **残差估计**（替代显式计算残差）
  - **秩剪枝（rank pruning）**（解耦块大小与最终秩）
  - **无伴随变体**（仅使用正向算子 $\mathbf{A}$）

- **关键技术细节**：

  ### A. 随机残差指示器（Frobenius范数）
  - 在第 $i$ 步，计算草图 $\mathbf{Y}_i = \mathbf{E}_{i-1} \mathbf{\Omega}_i$，其中 $\mathbf{\Omega}_i \in \mathbb{R}^{n \times b}$ 是标准高斯矩阵的归一化形式（$\sqrt{b}$ 倍缩放）。
  - 根据 Hutchinson 定理，$\mathbb{E}[\|\mathbf{Y}_i\|_F^2] = \|\mathbf{E}_{i-1}\|_F^2$，因此 $\|\mathbf{Y}_i\|_F$ 作为 $\|\mathbf{E}_{i-1}\|_F$ 的无偏估计。
  - 计算 $\mathbf{Y}_i$ 时无需存储残差：利用关联性 $\mathbf{E}_{i-1}\mathbf{\Omega}_i = \mathbf{A}\mathbf{\Omega}_i - \sum_{k=1}^{i-1} \mathbf{Q}_k(\mathbf{B}_k\mathbf{\Omega}_i)$，只需一次正向 matvec 和少量小矩阵乘法。
  - 避免了 `randQB_EI` 的Gramian减法导致的灾难性抵消，精度可达机器精度。

  ### B. 随机残差指示器（谱范数）
  - 使用“前向块” $\|\mathbf{B}_i\|_2$ 作为 $\|\mathbf{E}_{i-1}\|_2$ 的代理，其中 $\mathbf{B}_i = \mathbf{Q}_i^\top \mathbf{A}$。
  - 理论依据：$\mathbf{Q}_i$ 捕捉了 $\mathbf{E}_{i-1}$ 的主导左奇异子空间，因此 $\|\mathbf{B}_i\|_2 \approx \|\mathbf{E}_{i-1}\|_2$。
  - 可与自适应停止准则 $\|\mathbf{B}_i\|_2 \leq \varepsilon$ 结合，避免保守常数。

  ### C. 秩剪枝（Rank Pruning）
  - **问题**：大块大小 $b$ 可提高BLAS-3效率，但若容差在块中间满足，块大小会导致最终秩过剩 $b-1$。
  - **方法**：仅对最后一块 $\mathbf{Q}_{i-1}\mathbf{B}_{i-1}$ 操作：
    - **Frobenius范数**：由于 $\mathbf{Q}_{i-1}$ 列正交，$\|\mathbf{E}_{i-2} - \mathbf{Q}_{i-1}^s \mathbf{B}_{i-1}^s\|_F^2 = \|\mathbf{E}_{i-1}\|_F^2 + \|\mathbf{B}_{i-1}^r\|_F^2$。因此按 $\mathbf{B}_{i-1}$ 行范数升序丢弃，直到不满足容差。
    - **谱范数**：对 $\mathbf{B}_{i-1}$ 做SVD，保留奇异值 $\sigma_j \geq \varepsilon$ 的左右方向，丢弃小的。
  - 复杂度：$O(b n)$（Frobenius）/ $O(n b^2)$（谱），相对于matvec代价可忽略。

  ### D. 无伴随变体（Adjoint-Free）
  - 仅使用正向 $\mathbf{A}$，无需 $\mathbf{A}^\top$：
    - 残差草图：$\mathbf{Y}_i = \mathbf{E}_{i-1}\mathbf{\Omega}_i = \mathbf{A}\mathbf{\Omega}_i - \sum \mathbf{Q}_k (\mathbf{Q}_k^\top \mathbf{A}\mathbf{\Omega}_i)$，用 $\mathbf{Z}_i = \mathbf{A}\mathbf{\Omega}_i$ 替代 $\mathbf{B}_k$。
    - 秩剪枝：用 $\tilde{\mathbf{B}}_{i-1} = \mathbf{Q}_{i-1}^\top \mathbf{Z}_i$（即 $\mathbf{B}_{i-1}\mathbf{\Omega}_i$）的SVD估计 $\mathbf{B}_{i-1}$ 的奇异值（基于子空间嵌入性质）。
    - 谱范数估计：使用 $\|\mathbf{Y}_i\|_2$ 作为 $\|\mathbf{E}_{i-1}\|_2$ 的估计。

  ### E. 相对容差的矩阵自由估计
  - $\|\mathbf{A}\|_F$ 通过 $\|\mathbf{Z}_i\|_F$ 的无偏估计累积：$S_i = \sqrt{\frac{1}{i} \sum_{k=1}^i \|\mathbf{Z}_k\|_F^2}$。
  - $\|\mathbf{A}\|_2$ 通过拼接的草图 $[\mathbf{Z}_1\cdots\mathbf{Z}_i]$ 的谱范数估计（按 $\sqrt{i}$ 缩放）。

- **算法流程**（以 randQB_MF_Fro 为例）：
  1. 循环 $i=1,2,\dots$：
     - 采样 $\mathbf{\Omega}_i \sim \mathcal{N}(0,1/b)$
     - 计算 $\mathbf{Z}_i = \mathbf{A}\mathbf{\Omega}_i$（正向matvec）
     - 计算 $\mathbf{Y}_i = \mathbf{Z}_i - \sum_{k=1}^{i-1} \mathbf{Q}_k(\mathbf{B}_k\mathbf{\Omega}_i)$
     - 若 $\|\mathbf{Y}_i\|_F \leq \varepsilon$，则停止（最后一块为 $\mathbf{Q}_{i-1},\mathbf{B}_{i-1}$）
     - 否则，$\mathbf{Q}_i = \text{orth}(\mathbf{Y}_i)$，并做再正交化，计算 $\mathbf{B}_i = \mathbf{Q}_i^\top \mathbf{A}$（伴随matvec）
  2. 对最后一块 $\mathbf{B}_{i-1}$ 执行 `PruneRank_Fro`（按行范数），得到选择索引 $s$。
  3. 输出 $\mathbf{Q} = [\mathbf{Q}_1\dots\mathbf{Q}_{i-2}\ \mathbf{Q}_{i-1}^s]$，$\mathbf{B} = [\mathbf{B}_1^\top\dots\mathbf{B}_{i-2}^\top\ (\mathbf{B}_{i-1}^s)^\top]^\top$。

## 3. 实验设计

- **数据集/场景**：使用**合成矩阵**，大小为 $1024\times1024$，通过SVD定义 $\mathbf{A} = \mathbf{U}\mathbf{\Sigma}\mathbf{V}^\top$，其中 $\mathbf{U},\mathbf{V}$ 为随机正交矩阵，$\mathbf{\Sigma}$ 对角元设置为三种衰减模式：
  - 指数衰减：$\sigma_i = e^{-i/2}$
  - 多项式衰减：$\sigma_i = 1/i^p$（$p=4$ 用于Frobenius实验，$p=2$ 用于谱实验）
  - S形衰减：前400个常数，后624个按幂律下降至接近机器精度
- **对比方法**：
  - *Frobenius范数*：`randQB_EI`（SOTA自适应，但非无矩阵），`t-SVD`（最优但不可行）
  - *谱范数*：`randQB_HMT`（自适应范围查找器，仅正向算子），`t-SVD`
  - *提议方法*：`randQB_MF_Fro`、`randQB_AF_Fro`；`randQB_MF_Spec`、`randQB_AF_Spec`
- **评估指标**：
  - 输出秩 $k$ 与相对误差 $\|\mathbf{A}-\mathbf{Q}\mathbf{B}\|/\|\mathbf{A}\|$（根据范数类型）

- **参数设置**：块大小 $b=32$（指数矩阵用 $b=16$ 使迭代次数可比），容差 $\varepsilon$ 从 $10^{-1}$ 到 $10^{-15}$ 梯度变化。

## 6. 论文的主要结论与发现

- **Frobenius范数**：
  - `randQB_MF_Fro` 和 `randQB_AF_Fro` 达到的误差**严格低于或等于预设容差**，且精度可达到**机器精度**（$10^{-15}$ 量级）。
  - `randQB_EI` 在 $\varepsilon \lesssim 10^{-7}$ 时失效（精度墙），表现为误差指示器停滞或变为负值。
  - 提议方法的秩与最优SVD秩非常接近（通常仅多几维）。
- **谱范数**：
  - `randQB_MF_Spec` 使用前向块指示器 $\|\mathbf{B}_i\|_2$ 同样获得紧凑的秩，远优于 `randQB_HMT`（后者因保守常数导致秩大幅膨胀）。
  - `randQB_AF_Spec` 仅使用正向算子，秩稍高但误差仍能控制在同一数量级。
  - 误差指示器 $\|\mathbf{B}_i\|_2$ 与真实残差 $\|\mathbf{E}_{i-1}\|_2$ 在全动态范围内紧密匹配。
- **通用结论**：提出的算法族在无矩阵环境下实现了**自适应、高精度、秩近似最优**的低秩近似，同时保持计算效率。

- **实验覆盖不足**：
